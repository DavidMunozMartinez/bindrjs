If Binding: :if

The :if binding takes the provided expression and only cares if it evaluates truthy, if it does the content is renderer, if its falsy the content disappears entirely from the DOM

Its important to know that whatever bindings are in the contents of the elements that make use of the :if binding are not evaluated if the expression resolves to falsy, only when the expression is evaluated to truthy and the content is actually rendered, only then the children bindings are evaluated and computed
