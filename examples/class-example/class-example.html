Class Binding: :class

There are two main ways if binding classes to your HTML

  • Bind string property :class

    • Using the attribute you can bind any string defined in the Bind instance by providing it to the value

    Changes to the bind property attached to the attribute will immediately reflect and apply to the element

  • Conditional class name :class:class-name

    • Using the attribute you can provide an expression that if truthy it will apply the class-name provided in the second part of the attribute name
