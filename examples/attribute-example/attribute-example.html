Attribute Binding: :[attr]

You can bind any property defined in the Bind object to virtually any attribute by using the : prefix

A few examples:

  • <img :src="this.bindProp"/>
  • <a :href="this.bindProp"> </a>
  • <p :title="this.bindProp"> </p>
  • <input :checked="this.bindProp"/>

    • These will also react to data changes
