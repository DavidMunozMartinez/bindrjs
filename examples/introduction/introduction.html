BindrJS is a lightweight data reactivity library built to handle HTML data binding, the most efficient way possible, DOM updates are as close as possible to data updates thanks to native JavaScript Proxy API

In addition to the classic HTML string interpolation, it provides the most basic data binding types out of the box and virtually any attribute to help make your HTML powerful, dynamic and with the smoothest learning curve, mirroring vanilla JavaScript and HTML as much as possible

This library is also meant to be easily adoptable by ongoing projects that are not currently using any data-binding solutions, and require the least amount of setup as possible, making it an easy pick for quick small projects or to provide a simple data binding solution to large legacy projects

In addition to the out of the box binding types, you are able to add your own binding types and take advantage of the data reactivity system however you see fit

All you need to know is how to create HTML Bind Handlers You can create them with attributes that have a colon ( : ) before them or content inside ( ${this.interpolateText} )

For example:

Both the :class attribute and ${this.interpolateStart}this.bindValue${this.interpolateEnd} will become bind handlers.

Where the :class bind handler will depend on this.bindClass and ${this.interpolateStart}this.bindValue${this.interpolateEnd} depends on this.bindValue. And every time each handler dependency is updated, the HTML Bind Handler's will be re-computed
