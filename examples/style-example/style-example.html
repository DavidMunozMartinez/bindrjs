Style Binding :style

You can bind styles to your HTML with the :style attribute by providing an object as a value, each key in the object will represent a css property and the value will represent the property value, remember that multi-word property names needs to be camel cased to be correctly applied.

For example .class { background-color: red }

Needs to be defined in the object as let background = { backgroundColor: 'red' }

NOTE: Specific property binding like :style:property-name is still not supported
